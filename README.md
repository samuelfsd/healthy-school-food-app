# Plataforma Colaborativa - Merenda Escolar

## About
Uma aplicação web desenvolvida para coletar feedback dos alunos sobre a merenda escolar e visualizar os dados para auxiliar na melhoria nutricional. O projeto conta com um formulário de pesquisa detalhado e um dashboard de resultados com gráficos interativos em tempo real.

## Tech Stack
* **Frontend/Framework:** Next.js 16
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS
* **Banco de Dados:** MongoDB
* **ORM:** Prisma
* **Gráficos:** Recharts
* **Validação:** Zod & TanStack Form
